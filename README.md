# CLI based Trivia
 - This quiz is about CLI basics and consists of 4 questions, each with three answers to choose from.
 - The quiz requires the @inquirer prompt form node modules so make sure to have it installed through the command:
#npm install
- The command used to start the quiz on the CLI is 'trivia'
- to input the command into your CLI:
  chmod +x ./indextrivia.js
  npm link
  - then the trivia command can be used.
